Features

Our healthcare platform offers a wide range of features designed to improve patient care and streamline medical processes. Here are some of the key features:

  • Measures heartrate using phone camera
  • Environment factors like noise can be detected using microphone
  • Sleep cycle is tracked using screen time
  • Other health factors are collected using smart watches
  • Simple games are added to reduce anxiety
  • Easy to operate UI
